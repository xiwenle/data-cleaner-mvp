# 用户操作指南

> 欢迎使用数据清洗工具 Data Cleaner MVP！本文档将引导你完成从上传到下载的完整流程。
> 即使你没有任何编程经验，也能在 5 分钟内完成数据清洗。

---

## 界面概览

打开工具后，你会看到以下区域：

| 区域 | 位置 | 作用 |
|-----|-----|-----|
| **隐私提示** | 页面最顶部 | 数据安全说明，请务必阅读 |
| **侧边栏** | 页面左侧 | 调整清洗策略、手动指定列类型 |
| **上传区** | 页面中央 | 拖拽或点击上传 CSV/Excel 文件 |
| **预览区** | 上传区下方 | 查看数据前 20 行和统计摘要 |
| **检测结果** | 预览区下方 | 自动展示缺失值、重复行等 |
| **清洗按钮** | 检测结果下方 | 点击"一键清洗"开始工作 |
| **清洗报告** | 按钮下方（清洗后出现） | 完整的 HTML 报告 |
| **下载区** | 报告下方 | 下载 CSV 或 Excel |

## 逐步操作指引

### 第一步：上传文件

- 拖拽 CSV（.csv）或 Excel（.xlsx/.xls）文件到上传区域，或点击"Browse files"选择文件
- 文件上限 50MB，行数上限 10 万行。超出会提示拆分
- 支持 UTF-8 和 GBK 编码的 CSV；如果所有编码都无法识别，会提示你用 Excel 另存为 UTF-8

### 第二步：选择 Sheet（仅 Excel 多 Sheet 时）

- 如果你的 Excel 包含多个 Sheet，上传后会自动弹出下拉框
- 选择你要处理的 Sheet，页面会自动重新加载数据
- 单 Sheet 文件不需要此步骤

### 第三步：确认数据预览与列类型

- 系统自动展示前 20 行数据预览，你可以快速扫一眼确认数据是否正确
- **统计摘要**表格会显示每列的推断类型（数值/文本/日期/邮箱/手机号）、缺失数量、唯一值数量
- **如果列类型推断有误**（例如电话列被误判为数值）：展开左侧边栏的"列类型覆盖"，在对应列的下拉框中手动修改。比如把"phone"列改为"文本"或"手机"

### 第四步：查看自动检测结果

在清洗之前，系统已经帮你完成了初步检测：

- **缺失值检测**：每列缺失多少个数据项、缺失率多少
- **重复行检测**：有多少行是完全重复的
- **详细检测**（点击展开）：邮箱格式错误、手机号格式错误、数值异常值的位置和下界/上界

这些都是**只看不改**的，不会影响你的原始数据。

### 第五步：配置清洗策略

打开左侧边栏，你可以调整：

| 配置项 | 选项 | 说明 |
|-------|-----|-----|
| 删除完全重复的行 | 勾选/取消 | 勾选后保留每组重复的第一行，删除其余 |
| 数值列缺失值填充策略 | 自动/均值/中位数/填零/不填充 | "自动"会根据数据偏度选择中位数或均值 |
| 文本列填充值 | 自定义文字 | 留空=用该列出现最多的值（众数）；填文字=统一用这个值 |
| 检测并调整数值异常值 | 勾选/取消 | 勾选后用 IQR 方法找异常值 |
| IQR 倍数 | 1.0-3.0 | 越小越严格（更多异常值），越大越宽松 |

> 💡 **小贴士**：如果你拿不准，直接用默认配置即可。它们在大多数场景下效果很好。

### 第六步：点击"一键清洗"

- 点击蓝色大按钮"一键清洗"
- 看到旋转的加载动画即可等待
- 成功后页面跳转到清洗报告区域

### 第七步：查看清洗报告

报告以卡片形式展示，包含：

1. **数据变化摘要**：清洗前后的行数对比、缺失单元格数减少了多少
2. **清洗步骤详情**：每一步做了什么（占位符标准化、类型推断、去重、填充、异常值处理、格式标准化）
3. **缺失值填充详情**：每列用什么方式填充、填充了什么值、填了几个
4. **列类型识别结果**：每列被识别成什么类型、可信度
5. **警告与建议**：如果某列缺失率 >50% 会提醒你检查数据源；如果某列类型推断可信度低，建议手动确认

你可以通过"隐藏详细报告"复选框控制报告的显示/隐藏。

### 第八步：下载清洗后的文件

在报告下方有下载按钮：
- **下载清洗后 CSV**：通用格式，所有表格软件都能打开
- **下载清洗后 Excel**：.xlsx 格式，适合用 Excel 继续编辑

## 常见问题解答（FAQ）

### Q：为什么不支持身份证号？

身份证号、银行卡号等属于高风险敏感数据。出于隐私保护考虑，系统检测到这些列会自动阻止并提示你脱敏后再上传。如需处理这类数据，请下载源码在**本地**运行。

### Q：类型推断错了怎么办？

在左侧边栏找到"列类型覆盖"区域，在下拉框中将 `auto` 改为正确的类型即可。这个设置会在清洗时生效。

举个例子：你有一列"手机号"全是数字（如 `13812345678`），系统可能推断为"数值"。你手动改为"手机"类型后，系统就不会用均值去填充缺失值，而是用众数。

### Q：我的手机号是 `138-0000-1111`，为什么被检测出格式错误？

手机号格式检测前会先"归一化"：自动去除空格、横杠 `-`、加号 `+86` 等分隔符。归一化后 `138-0000-1111` 变成 `13800001111`（11 位数字，以 1 开头）是**合法**的。

但是如果你的手机号是 `12345`（不足 11 位）或 `1381234567`（只有 10 位），才会被标记为格式错误。

记得检查原始数据中是否有少位数的情况！

### Q：处理完数据去哪了？安全吗？

你的数据只在服务器内存中临时存在，处理完成后**不会被保存**到任何地方。几秒后内存自动释放。整个过程中数据不会离开我们的服务器，也不会被分享给第三方。

如果你处理的是高度敏感的数据，建议下载项目在**自己电脑上本地运行**。

### Q：可以处理更大的文件吗？

当前版本限制 50MB / 10 万行。如果你的文件超过这个限制：
1. 尝试拆分文件——比如按月拆分、按地区拆分
2. 在本地运行版本（`streamlit run app.py`）可以手动调整限制
3. 关注后续版本更新，我们计划逐步提高限制

### Q：清洗后数据变少了，正常吗？

**正常！** 如果原始数据有重复行，勾选"删除完全重复的行"后，重复部分会被删除，行数自然减少。

清洗报告会清楚地告诉你"删除了多少行"。如果你不想删除，可以在侧边栏取消勾选"删除完全重复的行"。

### Q：Excel 文件有多个 Sheet，怎么处理？

上传后会自动弹出一个下拉框，选择你要处理的 Sheet 即可。注意：每次只能处理一个 Sheet。如果需要处理多个 Sheet，需要分多次上传。
