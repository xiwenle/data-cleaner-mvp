> Data Cleaner MVP 的过去、现在与未来。

## ✅ 已完成（v1.0 MVP · 2025-07）

### 数据上传与预览
- [x] CSV 文件上传（自动检测 UTF-8/GBK 编码）
- [x] Excel 文件上传（.xlsx/.xls，多 Sheet 选择）
- [x] 文件大小限制（≤50MB）和行数限制（≤10万行）
- [x] 数据前 20 行预览
- [x] 统计摘要（列名、类型、缺失数、唯一值数）

### 核心清洗功能
- [x] 缺失值检测与填充（数值：auto/mean/median/zero/no_fill；文本：众数/自定义值）
- [x] 重复行检测与删除
- [x] IQR 异常值检测与裁剪（可调阈值 1.0-3.0）
- [x] 邮箱格式验证
- [x] 手机号格式验证与标准化（去除 +86、空格、横杠）
- [x] 列类型智能推断（数值/文本/日期/邮箱/手机号/ID）
- [x] 7 步清洗流水线（占位符标准化 → 类型推断 → 类型转换 → 去重 → 填充 → 异常值 → 标准化）

### 用户界面
- [x] macOS 风格 UI（克制的配色、充足留白、圆角 12px）
- [x] 侧边栏清洗配置（去重、填充策略、异常值阈值、文本填充值）
- [x] 侧边栏列类型手动覆盖
- [x] P0 检测结果展示（缺失值、重复行）
- [x] P1 可折叠检测结果（邮箱、手机号、异常值详情）
- [x] HTML 清洗报告（数据对比、步骤详情、填充详情、类型识别、警告建议）
- [x] CSV + Excel 双格式下载

### 隐私与安全
- [x] 页面上传组件前隐私警告
- [x] 敏感数据三级分级（高风险阻止、中风险确认、低风险放行）
- [x] 页脚法律免责声明
- [x] 数据不持久化存储
- [x] 新文件上传完整状态重置

### 测试
- [x] 48 个测试用例（17 validators + 19 cleaners + 10 happy path + 9 smoke）
- [x] 单元测试、集成测试全部通过，零警告
- [x] 核心流程功能验证 10/10 通过

## 🔜 短期计划（v1.1 · 预计 2025 Q3）

根据项目计划书 P1 功能清单：

| 优先级 | 功能 | 说明 |
|-------|-----|-----|
| 🟠 | 自定义验证规则 | 允许用户添加自己的验证规则（通过 UI 或简单配置） |
| 🟠 | 清洗历史记录 | 会话内保存最近 10 次清洗记录，可回溯查看 |
| 🟡 | 异常值阈值可视化 | 用箱线图展示异常值分布，比纯数字更直观 |
| 🟡 | 数据质量评分 | 清洗报告增加 0-100 的综合质量分 |
| 🟡 | 拖拽列排序 | 在预览界面支持列的重新排列 |
| 🟡 | 键鼠快捷操作 | Ctrl+Enter 触发清洗，Esc 关闭弹窗等 |
| 🟡 | 国际化支持 | i18n 框架，支持切换英文界面 |

### v1.1 技术优化

| 项目 | 说明 |
|-----|-----|
| 性能优化 | 对 >5 万行数据启用分块处理，降低内存峰值 |
| Benchmark 基准 | 建立清洗性能基准测试（1000/10000/100000 行） |
| 错误日志 | 增加结构化日志输出，便于排错 |
| CI/CD | GitHub Actions 自动运行 pytest |

## 📅 中期计划（v1.2 · 预计 2025 Q4 - 2026 Q1）

根据项目计划书 P2 功能清单：

| 功能 | 说明 | 复杂度 |
|-----|-----|-------|
| **数据库直连** | 支持从 MySQL、PostgreSQL 直接读取数据清洗 | 高 |
| **AI 修复建议** | 接入 LLM API，对复杂数据问题给出修复建议 | 中 |
| **多文件批量处理** | 同时清洗多个文件，支持文件夹上传 | 中 |
| **自定义清洗流水线** | 用户可拖拽编排清洗步骤和顺序 | 高 |
| **API 服务** | 提供 REST API 供其他系统调用 | 中 |
| **定时任务** | 定期自动清洗指定数据源并发送报告 | 中 |
| **数据版本对比** | 清洗前后数据对比视图，高亮差异 | 中 |
| **PDF 报告原生导出** | 使用 weasyprint 生成正式 PDF 报告 | 低 |

## 🚀 长期展望（v2.0+ · 2026+）

### 可能的商业化方向

| 方向 | 说明 | 示例 |
|-----|-----|-----|
| **垂直行业模板** | 预置电商/医疗/HR 等行业清洗模板 | 电商商品去重 + 价格异常检测模板 |
| **企业版 SaaS** | 多租户、权限管理、SSO 集成 | 按公司/团队隔离数据 |
| **私有化部署** | 提供 Docker 镜像 + 企业技术支持 | 银行、医疗机构客户 |
| **数据质量监控** | 持续监控数据源的实时清洗和分析 | 日/周报数据质量看板 |
| **AI 训练数据清洗** | 面向 ML 训练数据集的专用清洗工具 | 移除不均衡样本、标签清洗 |

### 社区贡献指引

- 欢迎提交 PR 修复 bug 或实现小的功能改进
- 大功能提议请先在 Issue 中讨论设计方向
- 遵循现有代码规范：`src/` 保持纯函数、测试覆盖新增代码
- 推荐新手从标记 `good first issue` 的任务入手
- 项目遵循 MIT 开源协议
